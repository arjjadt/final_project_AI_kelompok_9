Langkah 4: JALANKAN BAGIAN 4 (Main Simulation)

Langkah 5: JALANKAN BAGIAN 5 (Evaluation)

5. PEMECAHAN MASALAH (TROUBLESHOOTING)
------------------
- Masalah: "FileNotFoundError" untuk file peta.
  Solusi: Jalankan ulang cell "Download Data" untuk memastikan file mentah berhasil diambil.
- Masalah: "NameError" (misalnya, PIBT is not defined).
  Solusi: Pastikan Anda menjalankan cells secara berurutan dari atas ke bawah.
